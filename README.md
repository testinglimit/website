# Portfolio de Jesús Carmona

Web estática para portfolio de ilustración, cómic y diseño 2D/3D.

## Abrir la página

Abre `index.html` directamente en el navegador.

## Cambiar imágenes

Las imágenes de ejemplo están en `assets/`:

- `hero-carmona.png`: imagen principal.
- `comic-page-sample.png`: página de cómic destacada.
- `gallery-character.png`: dibujo de personaje.
- `gallery-guardian.png`: dibujo de escena.

Puedes sustituir esos archivos por imágenes reales manteniendo los mismos nombres, o añadir nuevas imágenes y actualizar las rutas en `index.html`.

## Cambiar contacto

El correo está al final de `index.html`:

```html
mailto:hola@jesuscarmona.art
```

Sustitúyelo por el email real de Jesús cuando lo tengas.
